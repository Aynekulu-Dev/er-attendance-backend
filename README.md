# Ethiopia Reads Attendance Backend

ይህ የኢትዮጵያ ሪድስ የቮለንቲየር መገኘት መከታተያ (Attendance Tracking) የባክኤንድ ሲስተም ነው።

## Technologies Used
- FastAPI
- SQLAlchemy (SQLite)
- Uvicorn

## Setup
1. Virtual environment ያዘጋጁ: `python -m venv venv`
2. Activate ያድርጉ: `source venv/bin/activate`
3. Requirements ይጫኑ: `pip install -r requirements.txt`
4. ሰርቨሩን ያስነሱ: `uvicorn app.main:app --reload`
